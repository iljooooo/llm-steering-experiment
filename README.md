# ACTIVATION ENGINEERING

:warning::traffic-light: WORK IN PROGRESS 

This repo was created rather quickly after I got insipired by these other two resources:
- Both [this video](https://www.youtube.com/watch?v=F9-u6Wezquk) and [its corresponding codebase](https://github.com/mensae/adv-steering)
- [This repo](https://github.com/nrimsky/CAA), published by members of the Anthropic R&D team.

The aim of this project is to replicate some of the experiments done in the literature, while also possibly expaning experimental evidences and creating a framework that could both allow for reproducibility and inclusion of future models with little changes to the codebase.

Also consider that this experiment is also used by myself to gain confidence in using some of the most common modern AI/ML frameworks, while also learning something new about modern steering techniques and similar.

I'm not yet sure about the direction that this project will take, so stay tuned for future updates!
